Settings file structure

File path : config/settings.ini

Description

This file is used to set the main settings of the application.

timer section

Settings about timers and delays.

Key Type Description Required/Optional Default
delay integer The delay between each requests group. Optional 5000

snmp section

Settings about SNMP.

Key Type Description Required/Optional Default
ip string The ip address of the device to reach. Optional "127.0.0.1"
timeout integer The timeout for SNMP requests. Optional 3000
autoretries integer Max number of autoretries for SNMP requests. Optional 10
filename string Path of the OIDs settings file. Optional "config/OID.ini"

mqtt section

Settings about SNMP.

Key Type Description Required/Optional Default
ip string The ip address of the device to send data. (MQTT server) Optional "127.0.0.1"
port integer The MQTT port. Optional 1883
